## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-ping`
- `allow-generate-secure-key`
- `allow-list-keys`
- `allow-sign-with-key`
- `allow-delete-key`
- `allow-check-secure-element-support`
